What does Ryu64 do?

Ryu64 emulates the Nintendo 64, the popular game system from 1996.

What Games does it run?

None at the time of writing this, it is very experimental, improvments to emulation and usability are coming very soon.

Shouldn't making an emulator for such a old system be easy?

No, not even close, it is very hard to make a Nintendo 64 emulator for many reasons. Any emulator, no matter the system, has its own set of challenges.

Why bother?

Emulators have been a passion of mine, so is programming, a good way to express that is to make an emulator.

Why the name "Ryu64"?

Simply put, the name just comes from the RyuJIT which is what .NET uses to run applications built in both Visual Basic and C#, we use the latter.

Can I communicate with you guys anywhere?

Sure, we have a Discord where everyone hangs out and talks about development or N64 emulation in general.
