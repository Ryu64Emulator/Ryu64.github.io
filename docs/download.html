Build your own

Dependencies

To build you need the .NET Core SDK
To clone the repository you need git, on Linux it should be as easy as using your package manager to install it, or it could already be installed. For Windows however you will need Git for Windows. (Make sure that "Git from the command line and also from 3rd party software" is checked in the setup)

Commands

To clone the master branch (from the directory you want to be the root Git directory):
git clone --recursive https://github.com/Ryu64Emulator/Ryu64.git ./
To run the master branch (from the root Git directory):
dotnet build -c Release
cd Ryu64
dotnet run -c Release -- "path/to/rom.z64"
To run feature branch in GUI mode (from the root Git directory):
dotnet build -c Release
cd Ryu64.GUI
dotnet run -c Release
To run feature branch in CLI mode (from the root Git directory):
dotnet build -c Release
cd Ryu64.CLI
dotnet run -c Release -- [options] "path/to/rom.z64"
